## How to compile and run the code?

# compile the code

STEP 1: 
COMPILE THE FILE:
```g++ ion.cpp -o ion```

STEP 2: 
PARSE THE CODE FILE:
```./ion {fileName}.ion```

    for example:
    ```./ion test.ion```

# run the code

STEP 3:
GENERATE THE CODE TO ASM:
```./fasm test.asm```

STEP 4:
RUN THE CODE:
``` ./test ```
